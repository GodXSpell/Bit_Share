Installation Instructions

Windows

  1. Download the ZIP file above
  2. Extract the contents to a folder
  3. Open PowerShell as Administrator
  4. Navigate to the extracted folder
  5. Run .\install.ps1

macOS

  1. Download the appropriate ZIP file for your Mac
  2. Extract the contents to a folder
  3. Open Terminal
  4. Navigate to the extracted folder
  5. Run chmod +x install.sh
  6. Run ./install.sh

Linux

  1. Download the TAR.GZ file above
  2. Extract it with tar -xzvf bitshare-linux-amd64.tar.gz
  3. Navigate to the extracted folder
  4. Run chmod +x install.sh
  5. Run ./install.sh

Running BitShare

After installation, you can run BitShare from any terminal:

bitshare

This will start BitShare in interactive mode.

Alternative Installation Methods

Package Managers

For macOS users with Homebrew:

brew tap yourusername/bitshare
brew install bitshare

For Windows users with Scoop:

scoop bucket add bitshare https://github.com/yourusername/scoop-bitshare
scoop install bitshare

For Developers

Clone the repository:

git clone https://github.com/yourusername/bitshare.git
cd bitshare
go build -o bitshare
